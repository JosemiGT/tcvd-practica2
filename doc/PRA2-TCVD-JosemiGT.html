3.2. Identificar y gestionar los valores extremos¶

Para obtener los valores extremos, vamos a buscar todos los registros que se encuentren al menos a 3 desviaciones estndar alejados de la media.

Definiremos la siguiente funcin para encontrarlos.

In [ ]:
def get_outliers(data_set, column_name, num_std):  
  
    column_mean = data_set[column_name].mean()  
    column_std = data_set[column_name].std()  
  
    column_upper_limit = column_mean + num_std * column_std  
    column_lower_limit = column_mean - num_std * column_std  
  
    # Contar la cantidad de valores extremos en la columna  
    return ((data_set[column_name] > column_upper_limit) | (data_set[column_name] < column_lower_limit))

4.2. Comparación de la normalidad y homogeneidad de la varianza¶

Para la comparacin de normalidad y homogeneidad de la varianza de las variables, vamos a utilizar la prueba de de normalidad de Shapiro-Wilk a las variables que vamos a emplear. Para ello definimos la siguiente funcin en Python.

In [ ]:
def shapiro_test(data_set, column_name):
    statistic, p_value = shapiro(data_set[column_name])  
    print("Prueba de normalidad de Shapiro-Wilk en la columna ", column_name, " :")  
    print("Estadstico de prueba:", statistic)  
    print("Valor p:", p_value)

4.3. Aplicación de pruebas estadísticas para comparar los grupos de datos¶

def wilcoxon_test(data_set, column_name_first, column_name_second):
    statistic, p_value = wilcoxon(data_set[column_name_first], data_set[column_name_second])  
    print("Prueba de Wilcoxon en las columnas ", column_name_first, " y ", column_name_second)  
    print("Estadstico de prueba:", statistic)  
    print("Valor p:", p_value)

def student_test(data_set, column_name, column_name_target):
    group1 = data_set[column_name][data_set[column_name_target] == 0]
    group2 = data_set[column_name][data_set[column_name_target] == 1]
    statistic, p_value = ttest_ind(group1, group2)  
    print("Test de student en las columnas ", column_name, " y ", column_name_target)  
    print("Estadstico de prueba:", statistic)  
    print("Valor p:", p_value)

6. Resolución del problema¶

A partir del anlisis anterior, hemos podido profundizar en qu variables mdicas tienen una mayor correlacin con probabilidad de infarto. Con esta informacin se puede intuir en un paciente las probabilidades de que sufra un infarto. Se observan que variables como la frecuencia cardiaca mxima alcanzada, el tipo de dolor torcico y la glucemia en ayunas mayor a 120 mg/dl pueden ser indicadores interesantes al tener una cierta correlacin. Por otro lado el resto de variables no encuentran diferencias estadsticas.

Por lo tanto, podemos concluir que de las variables analizadas, hay 3 que permiten darnos indicadores mdicos que puedan aproximar en cierta medida la probabilidad de infarto.
